HyperText Markup Language (HTML)

HTML is the standard markup language used to create Web pages. It defines the structure of the page through elements.

  • Container tags have opening and closing tags. Most elements have these.
  • Empty tags do not have closing tags.

All HTML elements have a default display value. The two most common are:

  • Block means the element always starts on a new line and takes up the full width available in the screen.
  • Inline means the element does not start on a new line and only takes the width of their content.

HTML Page Structure

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document Title</title>
</head>
<body>

</body>
</html>

The <a> tag allows for inserting links in the web page. The href attribute is used to indicate the destination of a link.
There are two kinds of links: relative and absolute. Relative links lead to a section in a web page, while absolute links lead to another website.

11 | Lists

There are three types of HTML lists: unordered, ordered, and description. Unordered and ordered lists have marks that can be changed using type for inline styling and list-style-type for external styling. <li> should be included to display the item.

  • Unordered lists are marked with discs, or the black circles by default. Some of its bullet styles are circle, square, and none. Customizing bullets can be done inline either in the <ul> or <li> or internally or externally by targetting those elements.
  • Ordered lists are marked with numbers by defalut. Some of its bullet styles are upper-roman, lower-roman, upper-alpha, and lower-alpha. The starting mark can be changed using the start attribute.
  • Description lists is for lists with terms, each with a description that is indented after the term. This uses <dt> and <dd> to display the term and the description respectively. 

<!--unordered list-->
<ul>
    <li>item</li>
    <li>item</li>
</ul>

<ul type="circle">
    <li>item</li>
    <li type="square">item</li>
    <li type="none">item</li>
</ul>

<!--ordered list-->
<ol>
    <li>item</li>
    <li>item</li>
    <li>item</li>
</ol>

<!--description list-->

Cascading Style Sheets (CSS)

CSS is the language used to style Web pages. It styles HTML elements by using selectors.
There are four types of CSS styles: inline, internal/embedded, external, and imported.

CSS Structure

Selector {
    Property: value;
    Property: value;
}

Inline CSS

Inline is located at the body of the HTML document. It is better used for styling specific and only a few HTML tags.

<p style="color: pink; text-align: center;">This is inline style.<p>

Internal/Embedded CSS

Internal/Embedded is located at the head of the HTML document. It is used for styling the whole document.

<style>
p {
    color: pink;
    text-align: center;
}
</style>

External CSS

External is a separate text file linked to the HTML document. This allows for shorter HTML document and better readability for both files. The link is written at the head of the HTML document. 

<head>
    <link rel="stylesheet" href="FileName.css">
</head>

Imported CSS

Imported is basically the same as external except this style allows for importing one style sheet into another.
Note! Some web creators use this style to hide their style sheets from viewers because older browsers cannot detect imports. If you want your web page design to work on all browsers, external style is recommended. 

@import ("FileName.css");
or 
@import url("FileName.css");

The Box Model

This wraps around every HTML element. It consists of the following:

  1. Content is where texts and images appear.
  2. Padding is the transparent area between the content and border.
  3. Border wraps around the padding and content.
  4. Margin is the transparent area around the border.

The sides of the padding, border, and margin can be adjusted specifically or in one line. The codes below are true to paddings, borders, and margins.

/*changing value specifically*/
{
margin-left: 10px;
margin-top: 5px;
margin-bottom: 15px;
margin-right: 10px;
} 
/*1 value: all sides are the same*/
{margin: 10px;} 
/*4 values: all sides will be changed*/
{margin: 10px 5px 15px 10px; /*order: top, right, bottom, left*/} 
/*3 values: for top, right-left, bottom*/
{margin: 20px 15px 5px;}

Selectors

Selectors target the elements to style them. There are different categories of selectors, but the most common one are the simple selectors which targets elements based on name, ID, and class.

  • Element Selector selects HTML elements based on the element name (body, h1, p, div...).
  • ID Selector selects HTML elements based on IDs or names preceded with a hashtag (#). This can only be assigned to a single element.
  • Class Selector selects HTML elements based on classes or names preceded with a period. This can be assigned to multiple elements.

Remember that spaces are not allowed for ID and class names.

/*element selector*/
p {
text-decoration: double;
}

/*id selector*/
#container {
width: 100%;
}

/*class selector*/
.section {
color: white;
}

Properties & Values

Properties are the attributes being styled and values are the setting of a property. The following are some of the common CSS properties and their values.

Width & Height

The width adjusts the horizontal size and the height adjusts the vertical size of a content.

img {
    width: 700px;
    height: 500px;
}

Color & Background Color

Color changes color of the foreground (usually texts) and background color changes the color of the background. Both accept different value formats such as name, RGB, and HEX. 

/*name value*/
.p1 {
    color: green;
    background-color: lightgreen;
}
/*rgb value*/
.p2 {
    color: rgb(0, 43, 128);
    background-color: rgb(144, 207, 238);
}
/*hex value*/
.p3 {
    color: #800071;
    background-color: #ee91c9;
}

Border & Border Radius

Border wraps the content in outlines and border radius adjusts the roundness of the corners of the border. The border's values consists of the thickness, the style, and the color. 

.border-ex1 {border: 5px dotted cornflowerblue;}
.border-ex2 {
    border: 2px solid blueviolet;
    border-radius: 10px;
}
/*4 values: for top-left, top-right, bot-right, bot-left*/
.four-values {
    border: 3px solid lightskyblue;
}
/*3 values: top-left, top-right-bot-left, bot-right*/
.three-values {
    border: 3px solid lightseagreen;
    border-radius: 10px 5px 30px;
}

Text Properties

These are attributes for texts. The following are the common text properties.

  • Text Align sets the horizontal alignment of the text. The default text alignment is left.
  • Text Decoration is the shorthand property of the following text decoration properties with some of their values:
    • text-decoration-line: underline, overline, line-through.
    • text-decoration-color is the color of the text decoration. It does not affect the color of the text itself.
    • text-decoration-style: solid (default), dotted, dashed, wavy.
    • text-decoration-thickness: any format of numbers.
  • Line Height is the vertical spacing of different lines of texts. The default line height is 1.2 or 120% although it also depends on the font family.

.line-height {line-height: 15px;}

.center {text-align: center;}

.right {text-align: right;}

.justify {text-align: justify;}

.text-decor {
    text-decoration: lightcoral overline wavy; /*text-decoration-
    color + text-decoration-line + text-decoration-style*/
}

Font Properties

These are attributes of the font of the text.

  • Font is the shorthand property for the following font properties with some of their values:
    • Font Family specifies the text's font.
      Note! It is recommened to have at least two fonts in case the first one cannot be read by the browser.
    • Font Size is the size of the font.
    • Font Weight is the boldness the font. The default font weight of headings is bold and other texts is normal. Some common values are: 100, light, 400, normal, 700, bold.
    • Font Style: normal (default), italic, bold, oblique.

p {
    font-family: Arial, Helvetica, sans-serif;
    font-size: 30px;
    font-weight: light;
    font-style: italic;
}

This text is 30 pixels big, in Arial font (with Helvetica as a substitute font) with light weight and italic style.
